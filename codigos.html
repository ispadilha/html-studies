Tags relacionadas a códigos:

HTML não é uma linguagem de programação, e sim uma linguagem de marcação.

As tags abaixo são usadas para marcar trechos de texto, atribuindo sentidos semânticos; relacionados a códigos e conceitos correlatos.

A tag "code":

É usada para designar trechos de código. O conteúdo é exibido em fonte padrão monospace.

É interessante usar juntamente com a tag de texto pré-formatado. Exemplo:

        
        x = 5;
        y = 6;
        z = x + y;

Tag de input de teclado:

É usada para expressar uma entrada de uma ou mais teclas. O conteúdo é exibido em fonte padrão monospace:

Ctrl + S

Tag de amostra de output:

Expressa uma amostra de saída de um programa computacional. O conteúdo é exibido em fonte padrão monospace:

File not found.
Press F1 to continue

Tag de variável:

É usada para designar variáveis de programas computacionais, ou de expressões matemáticas. O conteúdo geralmente é exibido em itálico. Exemplo:

The area of a triangle is: 1/2 * b * h, where b is the base, and h is the vertical height.
