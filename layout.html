Linhas horizontais:

Estas linhas que usarei para organizar os conteúdos, como esta logo abaixo, também são feitas com tags HTML.

Foram historicamente chamadas de "linhas horizontais", porque este é o padrão, mas podem ter esta orientação alterada usando CSS.

Links:

Esta tag permite incluir um link, como pode ser conferido clicando aqui.

Pode ser usada para transformar outros elementos em links, como por exemplo imagens, ao invés de texto.

Possui atributos para definir onde será aberta a página ou caminho de destino.

Os links podem conduzir a arquivos do proṕrio projeto, diferentes partes do documento, outros caminhos na internet, etc.

Podem também levar a envio de e-mail ou contato telefônico, mas estes casos ainda existem por razões históricas, e são pouco usados atualmente.

Existe uma tag HTML para criar botões:

No entanto, é importante saber que existem diferenças semânticas entre "âncoras" (links) e botões. Estes últimos podem ter outras funcionalidades que vão além de conduzir a um link. Habilitar ou submeter formulários, por exemplo.

Divisórias:

São um tipo básico e genérico para delimitação de conteúdo. São versáteis e não são semânticas (por isso "genéricas").

Este é um exemplo de div. Por ser genérica e versátil, pode envolver todo tipo de conteúdo, não apenas texto.

Iframes:

Esta é uma tag usada para exibir uma página dentro de outra página. Um iframe pode ser usado como alvo ("target") de um link.
