Linhas horizontais:

Estas linhas que usarei para organizar os conteúdos, como esta logo abaixo, também são feitas com tags HTML.

Foram historicamente chamadas de "linhas horizontais", porque este é o padrão, mas podem ter esta orientação alterada usando CSS.

"Divisórias"

São um tipo básico e genérico para delimitação de conteúdo. São versáteis e não são semânticas (por isso "genéricas").

Este é um exemplo de div. Por ser genérica e versátil, pode envolver todo tipo de conteúdo, não apenas texto.
