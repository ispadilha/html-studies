Formulários

São geralmente usados para receber dados do usuário, com o auxílio de diferentes elementos.

Estes podem ser: campos de texto, radio buttons, checkboxes, botão de submeter formulário, etc.

"Inputs"

São os elementos de formulários mais usados, e sintaticamente trata-se de uma tag versátil, cujo tipo é definido por um atributo.

É importante notar que eles precisam do atributo "name" para que seus dados sejam corretamente identificados ao serem enviados.

Uma exceção é o input do tipo "submit", que funciona sem o atributo "name", principalmente se for único.

Este tipo de input pode ter atributo "name", principalmente se houver mais de um, e seja o caso de identificar qual deles foi submetido ao servidor.

Rótulos

São usados juntamente com os campos de input em um formulário. Clicar no rótulo leva ao preenchimento do referido input.

São úteis em questão de acessibilidade, com softwares leitores de tela.

O atributo "for" do rótulo deve ser igual ao "id" do input a que se refere.

O input do tipo "submit":

Trata-se de um tipo particular, que serve para SUBMETER os dados preenchidos no formulário.

Geralmente aparece na forma de um botão.

Os dados são enviados a um "form handler", tipicamente um arquivo no servidor com um script para LIDAR com os dados recebidos.

O form-handler é especificado no atributo "action" do formulário.
