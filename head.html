Meta tags:

Sintaticamente, são diferentes atributos dentro de uma meta tag.

São usados por navegadores, motores de busca e outros sistemas.

Ficam dentro da tag head do documento, e especificam: set de caracteres, descrição da página, palavras-chave, autor do documento, viewport, etc.

Abaixo, seguem descrições de outras tags que são usadas dentro da tag head.

A tag script:

É uma tag específica para habilitar Javascript, conectando o arquivo html a um arquivo js.

É o que está permitindo a navegação entre diferentes páginas neste documento.

A tag base:

É uma tag que especifica um caminho padrão para links.
