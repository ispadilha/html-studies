Meta tags:

Sintaticamente, são diferentes atributos dentro de uma tag "meta".

São usados por navegadores, motores de busca e outros sistemas.

Ficam dentro da tag "head" do documento, e especificam: set de caracteres, descrição da página, palavras-chave, autor do documento, viewport, etc.

Abaixo, seguem descrições de outras tags que são usadas dentro da tag head.

A tag de título:

É onde se especifica o texto que aparece na aba do navegador.

É importante não confundir a tag "title" com o atributo "title", que é usado em diferentes tags para mostrar um tipo de "tooltip" ao hover do cursor.

A tag "link":

É usada para ligar outros arquivos ao documento em questão, como arquivos CSS para aplicar estilos, ou uma imagem para servir como "favicon".

A tag "script":

É uma tag específica para habilitar Javascript. Pode conectar o arquivo HTML a um arquivo JS, ou pode receber código Javascript diretamente.

É o que está permitindo a navegação entre diferentes páginas neste documento.

Pode ser usada dentro de uma destas tags: "head" ou "body" do documento. Esta decisão tem diferentes implicações, a serem aprofundadas em outros momentos e outros projetos, focados em Javascript ao invés de HTML.

A tag "style":

É usada para receber regras de CSS, escritas dentro da mesma, diretamente no documento, em vez de conectá-lo a um arquivo CSS externo. Este seria o caso descrito na tag "link", pois diferentemente da tag "script", a tag "style" não possui atributo para conectar um arquivo externo.

A tag "base":

É uma tag que especifica um caminho padrão para links.
