Linhas horizontais:

Estas linhas que usarei para organizar os conteúdos também são feitas com tags HTML.

Foram historicamente chamadas de "linhas horizontais", porque este é o padrão, mas podem ter esta orientação alterada usando CSS.

Quebras de linha:

É possível usá-las para ter maior espaçamento entre linhas de texto, como por exemplo entre os conceitos acima, ou aqui a seguir.


Estas quebras de linha também são feitas com uma tag HTML.
