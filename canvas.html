Canvas:

Esta tag é usada para criar elementos gráficos com o uso de Javascript.

Um elemento canvas é simplesmente uma área retangular numa página HTML, que por padrão não tem bordas nem conteúdo.

Para exibir os gráficos, é necessário que seja por Javascript.
