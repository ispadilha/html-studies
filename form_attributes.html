Atributos form*

Os exemplos abaixo têm como fonte o W3Schools.



O atributo "form":

Especifica a qual form pertence o input, para quando este não estiver dentro daquele, por algum motivo.





O input "formaction":

Especifica a URL do arquivo que irá processar a submissão do formulário, criando precedência sobre o atributo "action" do form.

Funciona com os seguintes tipos de input: submit, image.








O atributo "formenctype":

Especifica a forma de codificação dos dados do formulário, ao serem enviados. Cria precedência sobre o atributo "enctype" do form.

Destina-se apenas a forms com method="post".

Funciona com os seguintes tipos de input: submit, image.






O atributo "formmethod":

Define o método HTTP, para o envio dos dados do formulário, à action URL. Cria precedência sobre o atributo "method" do form.

Funciona com os seguintes tipos de input: submit, image.

Os dados podem ser enviados na URL (method="get") ou como uma transação HTTP post (method="post")

É importante saber que o método "get" adiciona dados à URL, que possui limite de tamanho, e não deve ser usado com informações sensíveis. Este método é útil para quando o usuário quer fazer um bookmark com o resultado, por exemplo.

O método "post" é mais robusto e seguro, e não tem limite de tamanho. Diferentemente do caso acima, não permite que se faça bookmark do resultado.








O atributo "formtarget":

Especifica onde será exibido o conteúdo referente à resposta após submissão do formulário. Cria precedência sobre o atributo "target" do form.

Funciona com os seguintes tipos de input: submit, image.








Os atributos "novalidate" e "formnovalidate":

Determinam que campos não deverão passar por validação quando o formulário for submetido. Para casos em que validação é feita de forma customizada em Javascript, por exemplo. Ou então quando validação não for necessária, como um submit de salvar rascunho, por exemplo.

"novalidate" é atributo de form, "formnovalidate" é atributo de input. Este cria precedência sobre aquele, e funciona apenas com input do tipo submit.
