Input do tipo "hidden":

É um tipo de campo de input que não é visível para usuários.

Permite que desenvolvedores incluam dados que não são visíveis ou alteráveis por usuários.

É bastante utilizado para especificar dados de destino de bancos de dados, por exemplo.

Os dados são visíveis e alteráveis em DevTools, por exemplo. Não deve-se utilizar com fins de segurança.

O input do tipo "submit":

Trata-se de um tipo particular, que serve para SUBMETER os dados preenchidos no formulário.

Geralmente aparece na forma de um botão.

Os dados são enviados a um "form handler", tipicamente um arquivo no servidor com um script para LIDAR com os dados recebidos.

O form-handler é especificado no atributo "action" do formulário.
