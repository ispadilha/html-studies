Outros casos e tipos de mídia:

Nesta seção, estudaremos outros tipos de mídia em relação a HTML.

Plug-ins:

São como pequenos programas que rodam dentro do navegador, extendendo suas funcionalidades.

Alguns exemplos de plug-ins antigos, e atualmente descontinuados:

  • Java Applets
  • ActiveX Controls
  • Shockwave Flash

E alguns exemplos atuais:

  • mapas
  • checagens de vírus
  • verificações de identidade
  • leitores de arquivos PDF

Objetos integrados

Não confundir com objetos em contexto de programação orientada a objetos.

Em HTML, objetos integrados são elementos com tags próprias, criadas com a finalidade de receber plug-ins.

Eles podem, no entanto, receber diversos outros tipos de elementos, como imagens, ou até outro arquivo HTML.

Existem duas tags distintas que podem ser usadas para estes elementos, exemplificadas abaixo.

YouTube

No momento de produção deste documento, o youtube é uma plataforma privada de grande alcance, que pode facilitar a incorporação de vídeos em HTML.

Isso é feito usando um elemento iframe, cujo atributo src recebe a URL do vídeo no youtube.

Este elemento poderá receber atributos e definições como:

  • reprodução automática
  • mutar a reprodução automática (isso é exigido por alguns navegadores)
  • definir uma playlist
  • reprodução em loop
  • exibir opções de controle

Pode não ser possível exibir ou reproduzir o conteúdo, devido a questões de segurança relacionadas ao conteúdo de um site incorporado em outro.

(Isso é possível mas possui exigências técnicas.)
