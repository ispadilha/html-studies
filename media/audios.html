Áudios:

De forma semelhante aos vídeos, as tags de áudio em HTML também recebem outras tags para especificar possíveis caminhos para arquivos do referido áudio.

O navegador irá utilizar o primeiro arquivo válido dentre os especificados.

Áudio de exemplo copiado do W3Schools.

A tag de áudio possui atributos importantes, para diferentes finalidades:

  • exibir botões e outros controles
  • determinar reprodução automática
  • mutar a reprodução automática, o que é exigido por alguns navegadores

    • (mutar para viabilizar a reprodução automática: é útil no caso de vídeos, mas áudios são praticamente inutilizados)

Formatos suportados:

São 3 os formatos de áudio suportados em elementos HTML:

  • mp3
  • wav
  • ogg

Elementos de áudio no DOM:

Assim como ocorre com elementos de vídeo, no DOM, elementos de áudio também ganham métodos, propriedades e eventos próprios. Isso permite que seus controles sejam manipulados por JavaScript também.
